## Linting

We use _eslint_ for syntax checking, and _Prettier_ for code formatting.

Eslint config: https://github.com/yannickcr/eslint-plugin-react

```
npm run lint
```

Prettier config: https://github.com/yannickcr/eslint-plugin-react

```
npm run format
```
